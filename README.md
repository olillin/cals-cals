## How to develop

### Prerequisites

You must have [Node.js®](https://nodejs.org/en/download/current) installed to start the website.

### Scripts

#### Start website

```console
npm run startup
```

#### Run tests

```console
npm test
```

#### Run formatter

```console
npm run format
```
